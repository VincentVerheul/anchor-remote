Help Anchor Remote

This web-app provides a remote control to the anchor windlass. The web-app is hosted on a Raspberri Pi micro computer which is connected to your anchor windlass.

Usage

{% if basic_mode %}
  • Set a target length based on the depth via the Set button. Optionally enter the name of the anchoring site when setting the target.
  • Use the Run button.
  • Use Pause during a run to pause the anchor action.
  • Use Resume to continue. The continuation direction (up or down) is shown next to the button.
  • When you have completed your anchoring operation, use Quit to shut down the Raspberri Pi micro computer before you power it off.
{% else %}
  • Set a target length either manually (via Adjust) or use the Set (target) option to set the target length based on the depth. Optionally enter the name of the anchoring site when setting the target.
  • Use the Run button.
  • Use Pause during a run to pause the anchor action.
  • Use Resume to continue. The continuation direction (up or down) is shown next to the button.
  • Before you initiate a run, you may adjust the target and current chain length and click the Adjust button.
  • To adjust manually, use the Go Up and Down buttons. The length adjusted is set by the slider; use the Adjust button. Use Pause to stop the manual run.
  • When you have completed your anchoring operation, use Quit to shut down the Raspberri Pi micro computer before you power it off.
{% endif %}

The History menu option lists events per anchoring site. The history is also shown when you click the anchor site hyperlink on the home page.

Target length going up

The target minimum length when pulling the anchor up is set to {{ min_length_up }} meter. This is a safety measure because the acual deployed chain length is only an estimate. Pull the anchor all the way up manually.

Configuration

You must specify some settings relevant to your boat. Select the Config menu and select your boat. Choose add boat when your boat (name) is not listed. Set the boat-independent anchor values and select the page layout: basic mode or full-featured. Adjust the time zone relative to your home time zone: positive going east, negative going west.

Use the Boat menu to specify your boat settings. Enter boat type (for information only) and boat name (for reference). Complete the other settings. Start with an estimated anchor down and up speed (in meter per minute) and carry out an anchor manouvre where you measure the actual (average) speeds. Update the settings with the new values.

The Theme menu option switches between a light and dark theme.

Raspberri Pi - WiFi

The Raspberri Pi (Rpi) connects to the up- and down controls of the windlass. The Rpi broadcasts a WiFi signal that you must select on your mobile phone. It is called 'AnchorRemote'. You will need a password to connect. Access the web application via the rpi5.local (rpi5 is the hostname of the Raspberri Pi).
